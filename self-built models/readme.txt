
load_datasets.py to load data.

ClassifyModel.py and DetectionModel.py are defined model structures

ClassificationTrain.py and DetectionTrain.py are files use models for training, the best performance is saved as the classfyModel.h5 and detectionModel.h5

ClassificationTest.py and DetectionTest.py use the saved models to test the test set.

To execute the codes, put all the files and the folder named dataset at the same directory.
